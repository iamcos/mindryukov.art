# Development Documentation

This folder contains all development standards, guides, and documentation for the mindrtukov project.

## Files

- **development-standards.md** - Overall development standards, project structure, and workflow
- **coding-conventions.md** - Code style, naming conventions, and best practices
- **extracted-context.md** - Key information extracted from chatlogs for website content
- **website-redesign-plan.md** - Design philosophy and implementation plan for the website

## Quick Reference

### Website Development
- Framework: Astro
- Styling: Tailwind CSS
- Location: `/website/`
- Start dev server: `cd website && npm run dev`

### Python Development
- Virtual environment: `.venv/`
- Activate: `source .venv/bin/activate`
- Location: Root directory

### Project Structure
- `/website/` - Astro website
- `/context/` - Context files and chatlogs
- `/mindryukov_films/` - Telegram channel data
- `/.cursor/` - Development documentation
- `/.venv/` - Python virtual environment
